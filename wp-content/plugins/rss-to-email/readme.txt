The RSS to Email plugin allows visitors to easily subscribe (for free) via email to blog updates using the RSS2Email.me service

== Description ==

[Rss to Email](http://rss2email.me/welcome/bloggers "RSS to Email Welcomes Bloggers") links your blog's RSS feed directly to the [rss2email.me](http://rss2email.me "rss2email.me") service so your readers can get daily email updates when you write new blog posts.  This is very convienient to readers because the service is free to start and allows them to combine several of their favorite blog updates into one daily email.  

You can read more

== Installation ==

To install RSS to Email, complete the following steps

1. Upload the folder `RSS to Email` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the Appearance -> Widgets Page and add the RSS to Email widget to the appropriate sidebar
4. Navigate to the Settings -> RSS to Email page and update the CSS style as required for your theme
